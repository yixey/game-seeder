Repack Features

  • Based on Batman.Episode.5-CODEX ISO release: codex-batman.episode.5.iso (12,890,767,360 bytes)
  • 100% Lossless but NOT MD5 Perfect: all game content is identical to original, but files were decrypted
  • Some files were replaced with ones from R.G. Mechanics repack (gratz, Spider91!) with another compression, also supported by game engine
  • NOTHING ripped, some game archives rebuilt
  • Significantly smaller archive size (compressed from 12 to 5.2 GB)
  • Installation takes from 15 minutes (8-cores CPU + SSD) up to one hour (2-cores CPU + HDD)
  • After-install integrity check so you could make sure that everything installed properly
  • HDD space after installation: 13 GB (~18 GB during installation)
  • Use “Language Selector.exe” in game root to change the game language
