Repack Features

  • Based on v1.05.5427.124 Origin release: 25 GB
  • RELOADED crack applied over
  • All single-player DLCs activated by default (you may need to press F12 once you’re on Normandya bridge; MP DLC bonuses can be activated by copying coalesced.bin file from _Unlocker folder to “BIOGame\CookedPCConsole”
  • 100% Lossless & MD5 Perfect: all files are identical to originals after installation (only when all languages and MP files installed)
  • NOTHING ripped, NOTHING re-encoded
  • Selective Download feature: you may skip downloading and installing of French/Italian/German files and Multiplayer DLC files (needed if you want to activate MP weapons in SP mode or install some specific mods)
  • Significantly smaller archive size (compressed from 25 to 8.2~11 GB, depending on selected components)
  • Installation takes: ~35 minutes on 8-threads CPU; ~1 hour 10 minutes on 4-threads CPU
  • After-install integrity check so you could make sure that everything installed properly
  • HDD space after installation: up to 25 GB (28 GB during installation)
  • Language can be changed by running corresponding .reg-file in “_Language Changer” folder
  • Repack uses XTool library by Razor12911
  • At least 2 GB of free RAM (inc. virtual) required for installing this repack
  • ATTENTION! If you’re installing this repack under Windows 10, run setup with compatibility set to Windows 7

If you see PhysX Installer error at the end of the installation, just close the message and run ut manually from _Redist folder.

FIFA 19 Repack Poll

December 2, 2018 FitGirl Leave a comment

As some of you could notice, I have a warning about FIFA 19 repack on my page. The thing is, that game resources are encrypted, thus normal compression is not possible. The only option is to split the game so you wouldn’t download commentary packs for languages you don’t need. The total saving is really tiny. The minimum size of such repack is 28.9 GB. But there is a bright sight too – it installs in mere 8-10 minutes from HDD to HDD. So, here’s my poll. The minimum required votes percentage for posting the repack in “as-is state” is qualified majority of 66%.

Mind you, that if the encryption problem will be solved, I will release a normal repack with proper compression, and not this crappy monkey job.
